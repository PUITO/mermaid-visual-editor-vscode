# 双栏编辑器使用指南

## 功能概述

对于不支持可视化拖拽编辑的图表类型，系统提供了双栏编辑器：
- **左侧**：SVG 实时预览（支持缩放和平移）
- **右侧**：Mermaid 代码编辑器（VS Code 风格）

## 支持的图表类型

以下图表类型使用双栏编辑器：

1. **classDiagram** - 类图
2. **stateDiagram** - 状态图
3. **gantt** - 甘特图
4. **pie** - 饼图
5. **requirementDiagram** - 需求图
6. **gitGraph** - Git 分支图
7. **journey** - 用户旅程图
8. **graph** - 基础流程图

## 界面布局

```
+------------------+------------------+
|                  |                  |
|   SVG 预览区      |   代码编辑器      |
|   (可拖动/缩放)   |   (VS Code 风格)  |
|                  |                  |
|                  |                  |
+------------------+------------------+
|  Scroll to zoom  |  Mermaid Code    |
|  Drag to pan     |                  |
|  Ctrl+0 reset    |                  |
+------------------+------------------+
```

## 交互操作

### SVG 预览区（左侧）

#### 缩放
- **滚轮向上**：放大（增加 10%）
- **滚轮向下**：缩小（减少 10%）
- **缩放范围**：10% - 500%
- **当前缩放比例**：显示在左上角工具栏

#### 平移
- **鼠标左键按住拖动**：平移视图
- **光标变化**：grab（可拖动）/ grabbing（拖动中）

#### 重置视图
- **按钮**：点击左上角的 "Reset" 按钮
- **快捷键**：Ctrl + 0（Windows/Linux）或 Cmd + 0（Mac）
- **效果**：恢复到 100% 缩放，居中显示

### 代码编辑器（右侧）

#### 编辑代码
- **直接输入**：在右侧编辑器中输入 Mermaid 代码
- **实时预览**：修改后左侧 SVG 自动更新（有防抖延迟）
- **语法高亮**：使用 VS Code 主题颜色

#### 键盘快捷键
- **Tab**：插入 4 个空格（可配置）
- **Enter**：换行
- **Ctrl/Cmd + Z**：撤销
- **Ctrl/Cmd + Y**：重做
- **Ctrl/Cmd + A**：全选

#### 自动保存
- 代码修改会自动同步到 VS Code 文档
- 无需手动保存
- 关闭文件时内容已保存

## 使用示例

### 示例 1：编辑类图

1. 打开 `tests/test-class.mmd`
2. 左侧显示类图的 SVG 渲染
3. 右侧显示 Mermaid 代码
4. 在右侧添加新的类定义：
   ```mermaid
   class NewClass {
       +String name
       +void method()
   }
   ```
5. 左侧立即显示更新后的类图
6. 可以拖动和缩放查看细节

### 示例 2：编辑状态图

1. 打开 `tests/test-state.mmd`
2. 在右侧代码中添加新状态：
   ```mermaid
   Processing --> Completed: 完成处理
   ```
3. 左侧 SVG 实时更新，显示新的状态转换
4. 使用滚轮放大查看标签文本

### 示例 3：编辑甘特图

1. 打开 `tests/test-gantt.mmd`
2. 修改任务日期或添加新任务
3. 左侧甘特图实时更新
4. 拖动查看完整的时间线

## 技巧提示

### 提高编辑效率

1. **分屏对比**
   - 保持左侧预览可见
   - 右侧专注编辑代码
   - 实时看到修改效果

2. **缩放查看细节**
   - 复杂图表使用放大功能
   - 查看小字体标签
   - 检查连接关系

3. **平移浏览大图**
   - 大型图表拖动查看不同区域
   - 无需滚动整个页面
   - 平滑的平移体验

4. **快速重置**
   - 迷失方向时使用 Ctrl+0
   - 快速回到初始视图
   - 或者点击 Reset 按钮

### 常见问题

**Q: 为什么左侧没有立即更新？**

A: 系统有 300ms 的防抖延迟，避免频繁重新渲染。输入完成后稍等即可看到更新。

**Q: 如何复制代码？**

A: 在右侧编辑器中使用 Ctrl+A 全选，然后 Ctrl+C 复制。

**Q: 缩放后如何回到原始大小？**

A: 按 Ctrl+0 或点击左上角的 Reset 按钮。

**Q: 可以调整左右栏的宽度吗？**

A: 当前版本固定为 50%/50%。未来版本可能支持拖动调整。

**Q: 代码有语法错误怎么办？**

A: 左侧会显示红色的错误提示框，说明具体的错误信息。修正代码后错误提示消失。

## 技术实现

### 组件架构

```
SplitEditor
├── Left Panel (SVG Preview)
│   ├── Toolbar (zoom level, reset button)
│   ├── SVG Container (pan & zoom)
│   └── Status Bar (hints)
└── Right Panel (Code Editor)
    ├── Toolbar (title)
    └── Textarea (VS Code styled)
```

### 关键特性

1. **平滑缩放**
   - CSS transform: scale()
   - transition 动画
   - 最小 10%，最大 500%

2. **流畅平移**
   - CSS transform: translate()
   - 鼠标事件监听
   - 拖动时光标变化

3. **实时同步**
   - onChange 事件
   - postMessage 到 VS Code
   - Mermaid 重新渲染

4. **主题适配**
   - 使用 VS Code CSS 变量
   - 自动适配深色/浅色主题
   - 一致的视觉体验

## 与可视化编辑器的对比

| 特性 | 双栏编辑器 | React Flow 编辑器 |
|------|-----------|------------------|
| 适用类型 | 类图、状态图等 | 流程图 |
| 编辑方式 | 代码编辑 | 拖拽节点 |
| 预览方式 | SVG 渲染 | Canvas 渲染 |
| 缩放支持 | 是（滚轮） | 是（内置控件） |
| 平移支持 | 是（拖动） | 是（拖动） |
| 实时预览 | 是 | 是 |
| 自动保存 | 是 | 是 |
